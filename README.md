# tic-tac-toe 

## how to start?

Для начала в **app/game_config.py** нужно выставить интересующие Вас настройки.

Если впервые скачали проект, пропишите:

```shell
python3 -m venv .venv  
```

```shell
source ./.venv/bin/activate
```

```shell
pip install psutil
```

```shell
python3 ./main.py
```

В следующие разы достаточно прописывать
```shell
source ./.venv/bin/activate
```

```shell
python3 ./main.py
```
